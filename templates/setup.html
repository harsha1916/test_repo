Alternative: Use Browser Console

You can also configure the API key using your browser console:

  1. Press F12 to open Developer Tools
  2. Go to the Console tab
  3. Run this command:
setApiKey('your-api-key-change-this')

Replace your-api-key-change-this with your actual API key from the .env file.
